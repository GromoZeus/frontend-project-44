# 🧠 Brain Games

## 📖 Описание проекта

«Brain Games» — это набор из пяти консольных игр, вдохновлённых популярными мобильными приложениями для тренировки мозга.  
В каждой игре нужно правильно отвечать на вопросы. После трёх правильных ответов игра завершается.  
❌ Неправильный ответ приводит к завершению игры и предложению сыграть снова.

## 🎮 Игры:

- **♾️ Brain even** — определение чётного числа.
- **🧮 Brain calculator** — вычисление случайных арифметических выражений.
- **🔢 Brain gcd** — определение наибольшего общего делителя.
- **📈 Brain progression** — поиск пропущенного числа в числовой последовательности.
- **🔍 Brain prime** — определение простого числа.

## ⚙️ Как установить

1. ✅ Убедитесь, что установлена **Node.js** версии 23 или выше.
2. 📦 Клонируйте репозиторий:
   
   `git clone git@github.com:GromoZeus/frontend-project-44.git`

3. 📂 Перейдите в директорию проекта:

   `cd frontend-project-44`

4. 📥 Установите пакеты:

   `npm install`
